# Data Table Component

Powerful table component with sorting, filtering, pagination, and row selection built on Table Primitive.

---

## Overview

The DataTable component provides a full-featured data table with sorting, filtering, pagination, row selection, and customizable columns. Built on the Table Primitive with shadcn/ui design.

**Features:**
- Column sorting
- Filtering
- Pagination
- Row selection
- Custom cell templates
- Responsive design
- Dark mode compatible

---

## Package

- **Package:** BlazorUI.Components
- **Namespace:** BlazorUI.Components.DataTable
- **Source:** https://github.com/blazorui-net/ui/tree/main/src/BlazorUI.Components/Components/DataTable

## API Reference

### `<DataTable>`

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `Data` | `IEnumerable<TData>` | **Required** | Data source |
| `State` | `TableState` | - | Table state (supports @bind-State) |
| `ChildContent` | `RenderFragment?` | `null` | Column definitions |

### `<DataTableColumn>`

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `Property` | `Expression<Func<TData, object>>` | **Required** | Data property |
| `Title` | `string?` | `null` | Column header title |
| `Sortable` | `bool` | `false` | Enable sorting |
| `Filterable` | `bool` | `false` | Enable filtering |
| `Template` | `RenderFragment<TData>?` | `null` | Custom cell template |

## Basic Usage

```razor
@using BlazorUI.Components.Pagination
@using BlazorUI.Primitives.Table

<DataTable Data="@users" @bind-State="tableState">
    <DataTableColumn TData="User" Property="u => u.Name" Title="Name" Sortable="true" />
    <DataTableColumn TData="User" Property="u => u.Email" Title="Email" />
    <DataTableColumn TData="User" Property="u => u.Role" Title="Role" />
</DataTable>

<Pagination State="@tableState.Pagination">
    <div class="flex w-full items-center justify-between py-4">
        <PaginationInfo />
        <PaginationContent>
            <PaginationItem>
                <PaginationFirst />
            </PaginationItem>
            <PaginationItem>
                <PaginationPrevious ShowText="false" />
            </PaginationItem>
            <PaginationItem>
                <PaginationPageDisplay />
            </PaginationItem>
            <PaginationItem>
                <PaginationNext ShowText="false" />
            </PaginationItem>
            <PaginationItem>
                <PaginationLast />
            </PaginationItem>
        </PaginationContent>
        <PaginationPageSizeSelector />
    </div>
</Pagination>

@code {
    private List<User> users = new();
    private TableState tableState = new();

    record User(string Name, string Email, string Role);
}
```

**Note:** DataTable uses the enhanced Pagination component instead of the deprecated TablePagination. The `TableState.Pagination` property provides a `PaginationState` that integrates with the Pagination component's automatic state management.
