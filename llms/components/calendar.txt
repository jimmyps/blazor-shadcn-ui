# Calendar Component

A date field component that allows users to select a date from a calendar view.

---

## Overview

The Calendar component provides an interactive calendar for date selection. It supports date constraints, disabled dates, configurable week start day, and various selection modes.

**Features:**
- Month and year can be quickly changed via dropdown selects in the header
- Configurable display of outside days (days from previous/next months)
- Keyboard navigation support
- Min/max date constraints
- Custom disabled dates function

## API Reference

### Main Component: `<Calendar>`

**Parameters:**

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `Selected` | `DateTime?` | `null` | Currently selected date |
| `SelectedChanged` | `EventCallback<DateTime?>` | - | Callback when selection changes |
| `MinDate` | `DateTime?` | `null` | Minimum selectable date |
| `MaxDate` | `DateTime?` | `null` | Maximum selectable date |
| `DisabledDates` | `Func<DateTime, bool>?` | `null` | Function to determine disabled dates |
| `FirstDayOfWeek` | `DayOfWeek` | `Sunday` | Which day starts the week |
| `ShowOutsideDays` | `bool` | `true` | Whether to show days from previous/next months (displayed faded and non-interactive) |
| `Class` | `string?` | `null` | Additional CSS classes |
| `OnSelect` | `EventCallback<DateTime>` | - | Callback when a date is selected |

## Basic Usage

```razor
<div class="border rounded-lg w-fit">
    <Calendar @bind-Selected="_selectedDate" />
</div>
<p>Selected: @(_selectedDate?.ToString("MMMM d, yyyy") ?? "None")</p>

@code {
    private DateTime? _selectedDate;

```

## Examples

### Example 1: Preselected Date

```razor
<Calendar @bind-Selected="_date" />

@code {
    private DateTime? _date = DateTime.Today;
}
```

### Example 2: Date Constraints (Min/Max)

```razor
<!-- Future dates only -->
<Calendar @bind-Selected="_futureDate" MinDate="@DateTime.Today" />

<!-- Past dates only -->
<Calendar @bind-Selected="_pastDate" MaxDate="@DateTime.Today" />

<!-- Date range (next 30 days) -->
<Calendar @bind-Selected="_rangeDate"
          MinDate="@DateTime.Today"
          MaxDate="@DateTime.Today.AddDays(30)" />
```

### Example 3: Disabled Dates (No Weekends)

```razor
<Calendar @bind-Selected="_date" DisabledDates="@IsWeekend" />

@code {
    private DateTime? _date;

    private bool IsWeekend(DateTime date)
    {
        return date.DayOfWeek == DayOfWeek.Saturday ||
               date.DayOfWeek == DayOfWeek.Sunday;
    }
}
```

### Example 4: Custom Disabled Dates

```razor
<Calendar @bind-Selected="_date" DisabledDates="@IsHoliday" />

@code {
    private DateTime? _date;
    private List<DateTime> holidays = new()
    {
        new DateTime(2025, 12, 25),
        new DateTime(2025, 1, 1),
    };

    private bool IsHoliday(DateTime date)
    {
        return holidays.Contains(date.Date);
    }
}
```

### Example 5: Week Start Day

```razor
<div class="flex gap-8">
    <div>
        <p class="text-sm mb-2">Sunday (default)</p>
        <Calendar @bind-Selected="_sundayDate" FirstDayOfWeek="DayOfWeek.Sunday" />
    </div>
    <div>
        <p class="text-sm mb-2">Monday</p>
        <Calendar @bind-Selected="_mondayDate" FirstDayOfWeek="DayOfWeek.Monday" />
    </div>
</div>

@code {
    private DateTime? _sundayDate;
    private DateTime? _mondayDate;
}
```

### Example 6: OnSelect Callback

```razor
<Calendar Selected="_date" OnSelect="HandleDateSelect" />
<p>Last selected: @_lastSelected</p>

@code {
    private DateTime? _date;
    private string _lastSelected = "None";

    private void HandleDateSelect(DateTime date)
    {
        _date = date;
        _lastSelected = date.ToString("yyyy-MM-dd");
    }
}
```

### Example 7: Hide Outside Days

Hide days from previous/next months for a cleaner appearance:

```razor
<Calendar @bind-Selected="_date" ShowOutsideDays="false" />

@code {
    private DateTime? _date;
}
```

## Accessibility

- Full keyboard navigation (arrow keys, Enter, Escape)
- ARIA labels for month navigation
- Screen reader support for date selection
- Focus management within calendar grid
- Disabled dates are announced to screen readers

---

## Common Patterns

### Booking Calendar

```razor
<Calendar @bind-Selected="_bookingDate"
          MinDate="@DateTime.Today"
          DisabledDates="@IsUnavailable" />

@code {
    private DateTime? _bookingDate;
    private List<DateTime> bookedDates = new();

    private bool IsUnavailable(DateTime date)
    {
        return bookedDates.Contains(date.Date) ||
               date.DayOfWeek == DayOfWeek.Sunday;
    }
}
```

### Birthday Picker

```razor
<Calendar @bind-Selected="_birthday"
          MaxDate="@DateTime.Today"
          MinDate="@DateTime.Today.AddYears(-120)" />
```
