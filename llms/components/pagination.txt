# Pagination Component

Page navigation with previous/next controls, page number links, and state-based automatic management.

---

## Overview

The Pagination component provides navigation controls for multi-page content. It includes previous/next buttons, first/last buttons, page number links, ellipsis indicators, page info display, and page size selection. Supports both manual control and automatic state management via `PaginationState`.

**Features:**
- Automatic state management with `PaginationState`
- First/Last page navigation buttons
- Page info display ("Showing X-Y of Z")
- Page number display ("Page X of Y")
- Page size selector dropdown
- Customizable text labels
- Manual control mode for custom implementations

**Properties:**

| Property | Type | Description |
|----------|------|-------------|
| `CurrentPage` | `int` | Current page number (1-based) |

| `CanGoNext` | `bool` | Whether next page exists |
| `CanGoPrevious` | `bool` | Whether previous page exists |
| `FirstItemIndex` | `int` | 1-based index of first item on page |

**Methods:**

| Method | Description |
|--------|-------------|
| `NextPage()` | Navigate to next page |
| `PreviousPage()` | Navigate to previous page |
| `FirstPage()` | Navigate to first page |
| `LastPage()` | Navigate to last page |

## Basic Usage

### With PaginationState (Recommended)

```razor
<Pagination State="@_paginationState" OnPageChanged="HandlePageChanged">
    <div class="flex w-full items-center justify-between">
        <PaginationInfo />
        <PaginationContent>
            <PaginationItem>
                <PaginationFirst />
            </PaginationItem>
            <PaginationItem>
                <PaginationPrevious ShowText="false" />
            </PaginationItem>
            <PaginationItem>
                <PaginationPageDisplay />
            </PaginationItem>
            <PaginationItem>
                <PaginationNext ShowText="false" />
            </PaginationItem>
            <PaginationItem>
                <PaginationLast />
            </PaginationItem>
        </PaginationContent>
        <PaginationPageSizeSelector />
    </div>
</Pagination>

@code {
    private PaginationState _paginationState = new() { TotalItems = 100, PageSize = 10 };

    private void HandlePageChanged(int page)
    {
        // Fetch data for new page
        Console.WriteLine($"Page changed to: {page}");
    }
}
```

### Manual Control

```razor
<Pagination>
    <PaginationContent>
        <PaginationItem>
            <PaginationPrevious OnClick="@(() => GoToPage(currentPage - 1))"
                                Disabled="@(currentPage <= 1)" />
        </PaginationItem>
        @for (int i = 1; i <= 5; i++)
        {
            var page = i;
            <PaginationItem>
                <PaginationLink IsActive="@(currentPage == page)"
                               OnClick="@(() => GoToPage(page))">
                    @(page)
                </PaginationLink>
            </PaginationItem>
        }
        <PaginationItem>
            <PaginationNext OnClick="@(() => GoToPage(currentPage + 1))"
                           Disabled="@(currentPage >= 5)" />
        </PaginationItem>
    </PaginationContent>
</Pagination>

@code {
    private int currentPage = 1;

    private void GoToPage(int page)
    {
        if (page >= 1 && page <= 5)
            currentPage = page;
    }
}
```

## Accessibility

- Uses `<nav role="navigation" aria-label="pagination">` for screen readers
- Previous/Next buttons have descriptive labels
- Current page has `aria-current="page"`
- Ellipsis is hidden from screen readers with `aria-hidden`
- First/Last buttons include screen reader text
