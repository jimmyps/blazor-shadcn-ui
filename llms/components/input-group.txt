# Input Group Component

Enhanced inputs with icons, buttons, and addons for complex input compositions.

---

## Overview

The InputGroup component enhances standard inputs by adding icons, buttons, and text addons before, after, above, or below the input field. Perfect for search boxes, URL inputs, code editors, and action-enabled fields.

**Features:**
- Icon, text, and button addons via InputGroupAddon
- Inline positioning (start/end of input)
- Block positioning (above/below input)
- Dedicated InputGroupInput and InputGroupTextarea components
- Support for multiple addons
- Dark mode compatible

## API Reference

### `<InputGroup>`

Root container for input groups.

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `Class` | `string?` | `null` | Additional CSS classes |
| `ChildContent` | `RenderFragment?` | `null` | Input and addons |

### `<InputGroupAddon>`

Wrapper for addon content (icons, text, buttons).

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `Align` | `InputGroupAlign` | Required | Position of addon |
| `Class` | `string?` | `null` | Additional CSS classes |
| `ChildContent` | `RenderFragment?` | `null` | Addon content |

### `InputGroupAlign` Enum

| Value | Description |
|-------|-------------|
| `InlineStart` | Before input (inline) |
| `InlineEnd` | After input (inline) |
| `BlockStart` | Above input (block layout) |
| `BlockEnd` | Below input (block layout) |

### `<InputGroupInput>`

Input field designed for InputGroup.

Inherits all standard input properties plus:
- `Type` - InputType enum
- `Placeholder` - Placeholder text
- `Value` - Input value
- `@bind-Value` - Two-way binding
- `AriaInvalid` - Validation state

### `<InputGroupTextarea>`

Textarea variant for InputGroup.

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `Rows` | `int?` | `null` | Number of rows |
| `Placeholder` | `string?` | `null` | Placeholder text |
| `Class` | `string?` | `null` | Additional CSS classes |

### `<InputGroupText>`

Text content for addons (currency symbols, labels, etc.).

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `Class` | `string?` | `null` | Additional CSS classes |
| `ChildContent` | `RenderFragment?` | `null` | Text content |

### `<InputGroupButton>`

Button for addons.

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `Variant` | `ButtonVariant?` | `null` | Button style variant |
| `Class` | `string?` | `null` | Additional CSS classes |
| `ChildContent` | `RenderFragment?` | `null` | Button content |

## Basic Usage

```razor
@using BlazorUI.Components.InputGroup

<InputGroup>
    <InputGroupAddon Align="InputGroupAlign.InlineStart">
        <LucideIcon Name="search" Size="16" />
    </InputGroupAddon>
    <InputGroupInput Placeholder="Search..." />
</InputGroup>
```

## Examples

### Search with Icon

```razor
<InputGroup>
    <InputGroupAddon Align="InputGroupAlign.InlineStart">
        <LucideIcon Name="search" Size="16" />
    </InputGroupAddon>
    <InputGroupInput
        Type="InputType.Search"
        @bind-Value="searchTerm"
        Placeholder="Search..." />
</InputGroup>

@code {
    private string searchTerm = "";
}
```

### Email with Validation Icon

```razor
<InputGroup>
    <InputGroupAddon Align="InputGroupAlign.InlineStart">
        <LucideIcon Name="at-sign" Size="16" />
    </InputGroupAddon>
    <InputGroupInput
        Type="InputType.Email"
        @bind-Value="email"
        Placeholder="name@example.com" />
    <InputGroupAddon Align="InputGroupAlign.InlineEnd">
        <LucideIcon Name="circle-check" Size="16" />
    </InputGroupAddon>
</InputGroup>

@code {
    private string email = "";
}
```

    </InputGroupAddon>

        @bind-Value="domain"

### Price Input with Currency

```razor

    <InputGroupAddon Align="InputGroupAlign.InlineStart">

    </InputGroupAddon>

```

### Search with Action Button

```razor
<InputGroup>
    <InputGroupInput Placeholder="Search products..." />
    <InputGroupAddon Align="InputGroupAlign.InlineEnd">
        <InputGroupButton Class="!px-2.5 !py-0.5 !rounded-sm">
            Search
        </InputGroupButton>
    </InputGroupAddon>
</InputGroup>
```

### Password with Toggle Visibility

```razor
<InputGroup>
    <InputGroupAddon Align="InputGroupAlign.InlineStart">
        <LucideIcon Name="lock" Size="16" />
    </InputGroupAddon>
    <InputGroupInput
        Type="@(showPassword ? InputType.Text : InputType.Password)"
        @bind-Value="password"
        Placeholder="Enter password" />
    <InputGroupAddon Align="InputGroupAlign.InlineEnd">
        <InputGroupButton
            Variant="ButtonVariant.Ghost"
            Class="!px-1.5 !py-1.5 !rounded-md"
            OnClick="@(() => showPassword = !showPassword)">
            <LucideIcon Name="@(showPassword ? "eye-off" : "eye")" Size="16" />
        </InputGroupButton>
    </InputGroupAddon>
</InputGroup>

@code {
    private string password = "";
    private bool showPassword = false;
}
```

### Copy to Clipboard

```razor
<InputGroup>
    <InputGroupInput Value="@shareUrl" />
    <InputGroupAddon Align="InputGroupAlign.InlineEnd">
        <InputGroupButton
            Variant="ButtonVariant.Ghost"
            Class="!px-1.5 !py-1.5 !rounded-md"
            OnClick="CopyToClipboard">
            <LucideIcon Name="copy" Size="16" />
        </InputGroupButton>
    </InputGroupAddon>
</InputGroup>

@code {
    private string shareUrl = "https://example.com/share/abc123";

    private async Task CopyToClipboard()
    {
        await JS.InvokeVoidAsync("navigator.clipboard.writeText", shareUrl);
    }
}
```

## Form Validation States

### Success State

```razor
<InputGroup>
    <InputGroupInput Value="johndoe" />
    <InputGroupAddon Align="InputGroupAlign.InlineEnd">
        <LucideIcon Name="circle-check" Size="16" Color="Green" />
    </InputGroupAddon>
</InputGroup>
<p class="text-sm text-green-600">Username is available</p>
```

### Error State

```razor
<InputGroup>
    <InputGroupInput
        Type="InputType.Email"
        Value="invalid-email"
        AriaInvalid="true" />
    <InputGroupAddon Align="InputGroupAlign.InlineEnd">
        <LucideIcon Name="triangle-alert" Size="16" Class="text-destructive" />
    </InputGroupAddon>
</InputGroup>
<p class="text-sm text-destructive">Please enter a valid email address</p>
```

### Loading State

```razor
<InputGroup>
    <InputGroupInput Placeholder="Enter code..." />
    <InputGroupAddon Align="InputGroupAlign.InlineEnd">
        <LucideIcon Name="loader-circle" Size="16" Class="animate-spin" />
    </InputGroupAddon>
</InputGroup>
<p class="text-sm text-muted-foreground">Verifying...</p>
```

## Styling Notes

- Use `Class="!px-2.5 !py-0.5 !rounded-sm"` on `InputGroupButton` for consistent sizing
- For icon-only buttons, use `Class="!px-1.5 !py-1.5 !rounded-md"`
- Add `Class="rounded-full"` to `InputGroup` for fully rounded inputs
- Use `order-*` classes with block addons to control visual order
