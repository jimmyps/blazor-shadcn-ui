# Hover Card Component

Rich preview card on hover with delay control and shadcn/ui design.

---

## Overview

The HoverCard component displays rich content previews on hover with configurable show/hide delays. Perfect for user profiles, link previews, and contextual information.

**Features:**
- Hover trigger
- Configurable delays
- Intelligent positioning
- Portal rendering
- ARIA attributes
- Dark mode compatible

---

## Package

- **Package:** BlazorUI.Components
- **Namespace:** BlazorUI.Components.HoverCard
- **Source:** https://github.com/blazorui-net/ui/tree/main/src/BlazorUI.Components/Components/HoverCard

**Note:** Requires `<PortalHost />` in your layout.

## API Reference

### Components

- `<HoverCard>` - Container
- `<HoverCardTrigger>` - Trigger element
- `<HoverCardContent>` - Card content

---

## Basic Usage

```razor
<HoverCard>
    <HoverCardTrigger AsChild>
        <a href="#" class="underline">@@username</a>
    </HoverCardTrigger>
    <HoverCardContent>
        <div class="flex gap-4">
            <Avatar>
                <AvatarImage Src="/avatar.jpg" />
                <AvatarFallback>JD</AvatarFallback>
            </Avatar>
            <div>
                <h4 class="text-sm font-semibold">John Doe</h4>
                <p class="text-sm text-muted-foreground">@@johndoe</p>
                <p class="text-xs mt-2">Software developer and open source contributor</p>
            </div>
        </div>
    </HoverCardContent>
</HoverCard>
```

---

## AsChild Pattern

Use `AsChild` on `HoverCardTrigger` to use your own element as the trigger instead of the default `<span>`:

### Button Trigger

```razor
<HoverCard>
    <HoverCardTrigger AsChild>
        <Button Variant="ButtonVariant.Link">@username</Button>
    </HoverCardTrigger>
    <HoverCardContent>
        <!-- User profile card -->
    </HoverCardContent>
</HoverCard>
```

### Link Trigger

```razor
<HoverCard>
    <HoverCardTrigger AsChild>
        <a href="/users/johndoe" class="font-medium underline">John Doe</a>
    </HoverCardTrigger>
    <HoverCardContent>
        <!-- User preview card -->
    </HoverCardContent>
</HoverCard>
```

**Note:** When `AsChild` is true, the child component automatically receives trigger behavior via `TriggerContext` including hover and focus event handling with configurable delays.
