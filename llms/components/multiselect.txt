# Multi Select Component

Searchable multi-select component with checkboxes, tags display, and Select All functionality.

---

## Overview

The Multi Select component allows selecting multiple options from a searchable dropdown. Selected items are displayed as dismissible tags with overflow indicator support.

**Features:**
- Multiple selection with checkboxes
- Search/filter functionality
- Select All with indeterminate state
- Selected items displayed as tags
- "+N more" overflow indicator
- Clear all functionality
- Form validation (EditContext)
- Keyboard navigation
- ARIA attributes
- Dark mode compatible

---

## API Reference

### Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| Items | IEnumerable<TItem> | Required | Collection of items to display |
| Values | IEnumerable<string>? | null | Currently selected values |
| ValuesChanged | EventCallback<IEnumerable<string>?> | - | Callback when selection changes |
| ValueSelector | Func<TItem, string> | Required | Function to extract value from item |
| DisplaySelector | Func<TItem, string> | Required | Function to extract display text |
| Placeholder | string | "Select items..." | Placeholder when empty |
| SearchPlaceholder | string | "Search..." | Search input placeholder |
| EmptyMessage | string | "No results found." | Message when no matches |
| ShowSelectAll | bool | true | Show Select All option |
| SelectAllLabel | string | "Select All" | Select All label text |
| MaxDisplayTags | int | 3 | Max tags before "+N more" |
| PopoverWidth | string | "w-[300px]" | Width of dropdown |
| Disabled | bool | false | Disable the component |
| Class | string? | null | Additional CSS classes |

## Basic Usage

```razor
@using BlazorUI.Components.MultiSelect

<MultiSelect TItem="Language"
             Items="languages"
             @bind-Values="selectedLanguages"
             ValueSelector="@(l => l.Value)"
             DisplaySelector="@(l => l.Label)"
             Placeholder="Select languages..."
             SearchPlaceholder="Search languages..." />

@code {
    private IEnumerable<string>? selectedLanguages;

    private List<Language> languages = new()
    {
        new() { Value = "csharp", Label = "C#" },
        new() { Value = "typescript", Label = "TypeScript" },
        new() { Value = "javascript", Label = "JavaScript" },
        new() { Value = "python", Label = "Python" }
    };

    public class Language
    {
        public required string Value { get; set; }
        public required string Label { get; set; }
    }
}
```

## Examples

### With Select All

```razor
<MultiSelect TItem="Technology"
             Items="technologies"
             @bind-Values="selectedTech"
             ValueSelector="@(t => t.Value)"
             DisplaySelector="@(t => t.Label)"
             Placeholder="Select technologies..."
             ShowSelectAll="true"
             SelectAllLabel="Select All"
             MaxDisplayTags="3" />

@code {
    private IEnumerable<string>? selectedTech;

    private List<Technology> technologies = new()
    {
        new() { Value = "react", Label = "React" },
        new() { Value = "vue", Label = "Vue.js" },
        new() { Value = "angular", Label = "Angular" },
        new() { Value = "blazor", Label = "Blazor" },
        new() { Value = "nextjs", Label = "Next.js" }
    };

    public class Technology
    {
        public required string Value { get; set; }
        public required string Label { get; set; }
    }
}
```

### Without Select All

```razor
<MultiSelect TItem="Framework"
             Items="frameworks"
             @bind-Values="selectedFrameworks"
             ValueSelector="@(f => f.Value)"
             DisplaySelector="@(f => f.Label)"
             Placeholder="Select frameworks..."
             ShowSelectAll="false" />
```

### In Form

```razor
<Field>
    <FieldLabel>Skills</FieldLabel>
    <FieldContent>
        <MultiSelect TItem="Skill"
                     Items="skills"
                     @bind-Values="model.SelectedSkills"
                     ValueSelector="@(s => s.Id)"
                     DisplaySelector="@(s => s.Name)"
                     Placeholder="Select your skills..."
                     ShowSelectAll="true"
                     PopoverWidth="w-[350px]" />
    </FieldContent>
    <FieldHelpText>Select all programming skills you have.</FieldHelpText>
</Field>
```

**Note:** Use specific widths (e.g., `w-[350px]`) rather than `w-full` since the popover renders via portal.

### Disabled State

```razor
<MultiSelect TItem="Language"
             Items="languages"
             Values="@(new[] { "csharp", "typescript" })"
             ValueSelector="@(l => l.Value)"
             DisplaySelector="@(l => l.Label)"
             Placeholder="Disabled multiselect"
             Disabled="true" />
```

## Keyboard Navigation

| Key | Action |
|-----|--------|
| Arrow Down | Move focus to next item |
| Arrow Up | Move focus to previous item |
| Space | Toggle checkbox |
| Enter | Toggle checkbox and close |
| Escape | Close dropdown |
